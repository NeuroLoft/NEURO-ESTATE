# Кейс: Технико-экономическое обоснование (ТЭО) проекта «Родовое поместье глазами нейросетей»

## Описание
Комплексный кейс для МГУ, Минэкономразвития, инвесторов и правительства региона. Включает:
- Основной документ ТЭО (структура — 50+ стр.)
- Подробный план разделов
- Приложения: библиотека технологических решений, сравнительные таблицы, шаблоны документов, нормативные акты, методики расчетов, бизнес-процессы

## Структура кейса
- ТЭО_Родовое_Поместье.docx — основной документ
- ТЭО_Родовое_Поместье_План_разделов.md — структура и план
- Приложения_Библиотека_Технологий/ — приложения (каталоги, таблицы, шаблоны, стандарты, методики)

## Особенности
- Сухой академический стиль
- Фокус на KPI, финансовых моделях, правовых схемах
- Сравнительный анализ с зарубежными аналогами
- Максимальная насыщенность приложений

## Цель
Создание эталонного ТЭО для экспертного и академического сообщества, инвесторов и органов власти.
