# Методика расчета KPI и финансовых моделей для проектов родовых поместий

## 1. Ключевые показатели эффективности (KPI)
- ROI (окупаемость инвестиций)
- IRR (внутренняя норма доходности)
- NPV (чистая приведённая стоимость)
- Payback Period (срок окупаемости)
- Социальный эффект (количество созданных рабочих мест, прирост населения)
- Экологический эффект (снижение выбросов, доля ВИЭ)

## 2. Финансовая модель
- Оценка капитальных затрат (CAPEX)
- Оценка операционных затрат (OPEX)
- Прогнозирование доходов (продажа продукции, аренда, услуги)
- Моделирование сценариев (оптимистичный, базовый, пессимистичный)
- Чувствительность к макроэкономическим факторам (инфляция, ставки, субсидии)

## 3. Пример расчета (формулы)
- NPV = Σ (CFt / (1 + r)^t) - IC
- IRR — ставка, при которой NPV = 0
- Payback Period = IC / Среднегодовой чистый доход

## 4. Инструменты и источники данных
- Excel, специализированные финансовые калькуляторы
- Открытые базы данных (Росстат, Минэкономразвития, международные источники)
